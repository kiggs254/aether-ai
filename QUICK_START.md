## What You Need to Do

### 1. Generate Widget Files (One Time)

```bash
npm run generate-widget
```

This creates:
- `public/widget.css`
- `public/widget.js`

### 2. Upload to Supabase Storage

**Go to Supabase Dashboard:**
1. Open your Supabase project
2. Click **Storage** in the sidebar
3. Click **New bucket**
4. Name it: `widget-assets`
5. Make it **Public** âœ… (important!)
6. Click **Create bucket**

**Upload Files:**
1. Click on `widget-assets` bucket
2. Click **Upload file**
3. Upload `public/widget.css`
4. Upload `public/widget.js`

**Get URLs:**
After uploading, the files will be at:
- CSS: `https://YOUR-PROJECT.supabase.co/storage/v1/object/public/widget-assets/widget.css`
- JS: `https://YOUR-PROJECT.supabase.co/storage/v1/object/public/widget-assets/widget.js`

### 3. That's It! âœ…

The embed script is already configured to use:
- Supabase Storage for `widget.css` and `widget.js`
- Supabase Edge Function for API calls (no credentials needed!)
