Widget Embed Test

This page tests the widget embed script. You should see a floating button in the bottom right corner.

Expected behavior:

  • A floating button should appear in the bottom right corner
  • Clicking the button should open the chat widget
  • The widget should display the welcome message
  • You should be able to type and send messages

Note: The widget will try to connect to the Supabase proxy function. If the URL/key are not configured, API calls will fail, but the widget UI should still be visible.
